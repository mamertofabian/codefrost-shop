# Codefrost Shop

This is a complete eCommerce web application developed from scratch.

# Technologies used
- MERN Stack (MongoDB, ExpressJS, ReactJS, and NodeJS)
- Redux
- Typescript on Frontend
- Paypal Integration (Stripe will be added soon)
